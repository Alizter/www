Use OPAM

OPAM is a package manager for OCaml, which can be used to install Coq and many of its packages. The general documentation can be found on the OPAM website. For example, to install the version 1.2.2 of OPAM from the sources on a Unix system:

curl -L https://github.com/ocaml/opam/archive/1.2.2.tar.gz |tar -xz
cd opam-1.2.2
./configure
make lib-ext
make
sudo make install

To add the repository for the Coq packages:

opam repo add coq-released https://coq.inria.fr/opam/released

To list all the version of Coq:

opam show --field=available-versions coq

To install the version 8.4.6 of Coq:

opam pin add coq 8.4.6

To install a package:

opam install coq-compcert

To list all the Coq packages:

opam search coq

To install the beta version of Coq:

opam repo add coq-core-dev https://coq.inria.fr/opam/core-dev
opam pin add coq 8.5~beta3
Make a package

New packages can be submitted by a pull-request on the opam-coq-archive repository. For more information, you can follow this tutorial or read the OPAM documentation.
