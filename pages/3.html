  Binaries for Mac OS X

For an installation via fink , you have to move package description files to /sw/fink/10.*/local/main/finkinfo/, and binary packages (optional) to /sw/fink/10.*/local/main/binary-darwin-i386/. Then type fink install coq or fink install coqide.
