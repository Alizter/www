This is a major evolution of the Coq system (see ANNONCE).

  •   Binaries for Linux

  •   Binaries for FreeBSD

  •   Binaries for Mac OS X
    • Main Coq package (without CoqIDE, to be typically used with Proof General)
      • Self installer for Intel Mac

    • Via Fink:
      • Package description files for Coq and for CoqIDE
      • Binary packages for Coq and for CoqIDE
        • Package description files go to /sw/fink/10.*/local/main/finkinfo/, binary packages (optional) to /sw/fink/10.*/local/main/binary-darwin-i386/. Then type fink install coq or fink install coqide.
  •   Binary distribution for Windows
    • Self installer for Coq and CoqIDE

    • The previous versions of Coq

    Most of the previous version of Coq are available online, via the ftp server. Whether you are looking for the V5.10 or the bleeding edge distribution V8.1, you will find it there.

    Development version of Coq

    For the impatient, the development version of Coq is available from our Subversion repository: 

    svn checkout svn://scm.gforge.inria.fr/svn/coq/trunk

    Warning: the working sources you can access through this server have not been tested as thoroughly as official releases. The programs obtained by compiling these sources may not work properly. The sources may even fail to compile at all. New features appearing in these working sources are generally undocumented yet, and may be removed or changed in incompatible ways at any time. So, you are most welcome to test those working sources, but please do not depend too much on them.
