== What is Coq ? == Coq is a formal proof management system. It provides a formal language to write mathematical definitions, theorems and proofs, which are mechanicaly checked by the machine. [[http://logical.futurs.inria.fr/coq/?q=node/6 More about Coq]] == How to get it ? == The latest version of Coq is 8.1. It has been released on february, 2007. You can get it by visiting our [http://logical.futurs.inria.fr/coq/?q=node/3 download] page. == Resources == Coq provides a huge set of commands and tactics, and comes with a rich standard library. To know more about it, please consider reading the [http://coq.inria.fr/V8.1/refman/index.html reference manual] and browse the whole [http://coq.inria.fr/V8.1/stdlib/index.html standard library]. If you want to take a glimpse at various formalizations made by the users of Coq, you can pay a visit to the [http://coq.inria.fr/contribs-eng.html contributions] page.
